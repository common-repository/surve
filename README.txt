== Description ==

Installeer de Surve plugin om snel de implementatie van de Surve-tracking code in uw website af te ronden.

Heeft u nog geen account? Meld u dan aan op Surve.nl om te kunnen beginnen.

Surve biedt uitgebreide Heatmaps, Feedbackbuttons en Surveys voor uw website.

== Installation ==

1. Plaats alle bestanden van de plugin in de map "surve" in uw plugins map van uw Wordpress installatie.

2. Activeer de plugin in uw Dashboard bij Plugins.

3. Vul het website ID in op de instellingen pagina, uw Surve code wordt automatisch aan uw website toegevoegd.
